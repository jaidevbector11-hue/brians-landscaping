# Brian's Landscaping Website

Static SEO website for Brian's Landscaping in Bethlehem, PA.

## Publish on GitHub Pages

1. Create a new public GitHub repository named `brians-landscaping`.
2. Upload all files and folders from this directory to the repository root.
3. In GitHub, open Settings > Pages.
4. Under Build and deployment, choose:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /root
5. Save. GitHub will publish the site at:
   `https://YOUR-USERNAME.github.io/brians-landscaping/`

## Before launch

- Replace `https://example.com` in meta tags, schema, robots.txt, and sitemap.xml with the final website URL.
- Replace placeholder SVG gallery images with real project photos.
- Confirm the actual business hours.
- Connect quote/contact/booking forms to a form backend, email/SMS notifications, and Google Calendar if desired.
